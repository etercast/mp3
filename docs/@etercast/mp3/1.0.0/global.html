    EncoderOptions()

    MP3 encoder options

    Properties:
    Name Type Description
    sampleRate number

    Input/output sample rate. Usually this is the value from an AudioContext.
    numChannels number

    Number of input/output channels. MP3 supports 1 (mono) or 2 (stereo) channels
    quality number

    Encoding quality (0 - lowest, 9 - highest). In VBR (Variable Bit-Rate) this quality indicates an average kbps but in CBR (Constant Bit-Rate) 0 is equal to 32kbps and 9 is equal to 320kbps.
    samples number

    Number of samples that will be encoded each time encode is called.
    mode EncoderMode

    Encoding mode (0 - CBR, 1 - VBR).
